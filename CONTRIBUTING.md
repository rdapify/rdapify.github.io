# Contributing to RDAPify Website

Thank you for your interest in contributing to the RDAPify website!

## ⚠️ Important Notice

**This repository contains only the built/deployed website files.**

All website source code and documentation are in the main repository:
- **Main Repository**: https://github.com/rdapify/RDAPify
- **Website Source**: https://github.com/rdapify/RDAPify/tree/main/website
- **Documentation**: https://github.com/rdapify/RDAPify/tree/main/docs

## How to Contribute

### For Website Changes

1. Fork the main repository: https://github.com/rdapify/RDAPify
2. Make changes in the `website/` directory
3. Test locally:
   ```bash
   cd website
   npm install
   npm start
   ```
4. Submit a pull request to the main repository

### For Documentation Changes

1. Fork the main repository: https://github.com/rdapify/RDAPify
2. Make changes in the `docs/` directory
3. Test locally (same as above)
4. Submit a pull request to the main repository

### For This Repository

**Do NOT submit pull requests to this repository** unless you're fixing:
- `CNAME` file
- `README.md`
- `LICENSE`
- This `CONTRIBUTING.md` file

All other files are automatically generated and deployed by GitHub Actions.
