# rdapify.github.io

الموقع الرسمي ومركز التوثيق لمشروع RDAPify.

## نظرة عامة

يحتوي هذا المستودع على الموقع الرسمي لـ RDAPify، وهي مكتبة عميل RDAP سريعة وآمنة متاحة في لغات برمجية متعددة:

- **TypeScript/JavaScript** (npm: `rdapify`)
- **Rust** (crates.io: `rdapify`)
- **Node.js Native** (npm: `rdapify-nd`)
- **Python** (PyPI: `rdapify-py`)
- **Go** (bindings: `rdapify-go`)


## التطوير المحلي

الموقع مبني بـ HTML/CSS/JavaScript ثابت ومستضاف على GitHub Pages.

### التطوير المحلي

```bash
# لا حاجة لعملية البناء — الملفات تُخدم كما هي
# افتح index.html في المتصفح لعرض الموقع
open index.html
```

### النشر

ينتشر الموقع تلقائياً عند دفع التغييرات إلى فرع `main`.

## إرشادات المحتوى

### للمحتوى الإنجليزي
- الدقة التقنية حتمية
- استخدم لغة واضحة واحترافية
- قم بتضمين أمثلة كود حيث ينطبق
- ربط إلى التوثيق ذي الصلة

### للمحتوى العربي
- استخدم اللغة العربية الفصحى المبسطة
- حافظ على الاتساق مع النسخة الإنجليزية
- الشروط التقنية تبقى بالإنجليزية (API و RDAP و webhook وما إلى ذلك)
- التنسيق من اليمين إلى اليسار (RTL) يطبق تلقائياً

### للغات أخرى
- حافظ على التكافؤ مع التوثيق الإنجليزي
- استخدم اللغة الرسمية المناسبة
- احفظ أمثلة الكود بالإنجليزية
