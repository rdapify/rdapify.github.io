# rdapify.github.io

Official website and documentation hub for the RDAPify project.

## Overview

This repository contains the official website for RDAPify, a fast, secure RDAP client library available in multiple languages:

- **TypeScript/JavaScript** (npm: `rdapify`)
- **Rust** (crates.io: `rdapify`)
- **Node.js Native** (npm: `rdapify-nd`)
- **Python** (PyPI: `rdapify-py`)
- **Go** (bindings: `rdapify-go`)

## Content Guidelines

### For English Content
- Technical accuracy is critical
- Use clear, professional language
- Include code examples where applicable
- Link to relevant documentation

### For Arabic Content (العربية)
- Use formal Modern Standard Arabic (فصحى مبسّطة)
- Maintain consistency with English version
- Technical terms remain in English (API, RDAP, webhook, etc.)
- Right-to-left (RTL) formatting applied automatically

### For Other Languages
- Maintain parity with English documentation
- Use appropriate formal language
- Preserve code examples in English
