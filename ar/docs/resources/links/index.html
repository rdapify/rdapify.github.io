روابط وموارد مفيدة

الغرض: مجموعة منتقاة من الموارد الخارجية الأساسية والمواصفات والأدوات والمجتمعات ذات الصلة بـ RDAP والبنية التحتية للإنترنت والأمان والامتثال لمطوري ومستخدمي RDAPify ذات صلة: مراجع RFC | قاموس المصطلحات | الأوراق البحثية | المواصفات وقت القراءة: 3 دقائق

بروتوكول RDAP والمعايير​

المواصفات الرسمية​

  • RFC 7480: استخدام HTTP في بروتوكول الوصول إلى بيانات التسجيل (RDAP) — RFC التأسيسي الذي يحدد بروتوكول RDAP واستخدام HTTP
  • RFC 7481: خدمات الأمان لبروتوكول الوصول إلى بيانات التسجيل (RDAP) — اعتبارات الأمان ومتطلبات تطبيقات RDAP
  • RFC 7482: تنسيق استعلام بروتوكول الوصول إلى بيانات التسجيل (RDAP) — مواصفات تنسيق الاستعلام التفصيلية
  • RFC 7483: استجابات JSON لبروتوكول الوصول إلى بيانات التسجيل (RDAP) — تعريفات تنسيق الاستجابة القياسية
  • RFC 8521: امتدادات استعلام بروتوكول الوصول إلى بيانات التسجيل (RDAP) — آليات الامتداد للاستعلامات المتقدمة

موارد IANA​

  • سجل خدمة تمهيد RDAP الخاص بـ IANA — نقاط نهاية خدمة التمهيد الرسمية لجميع امتدادات النطاقات العلوية
  • سجلات RDAP الخاصة بـ IANA — السجل الكامل لنقاط نهاية خدمة RDAP
  • مواصفات خدمة تمهيد RDAP — التفاصيل التقنية لتطبيق خدمة التمهيد

موارد الأمان والخصوصية​

الأمان المحدد لـ RDAP​

  • ضوابط أمان NIST SP 800-53 — إطار شامل لضوابط الأمان قابل للتطبيق على تطبيقات RDAP
  • RFC 9048: مصادقة استعلام RDAP — آليات المصادقة لاستعلامات RDAP
  • OWASP ASVS 4.0 — معيار التحقق من أمان التطبيقات مع أقسام ذات صلة بـ RDAP

الخصوصية والامتثال​

  • النص الرسمي لـ GDPR — النص الكامل للائحة العامة لحماية البيانات والتوجيه
  • النص الرسمي لـ CCPA — موارد قانون خصوصية المستهلك في كاليفورنيا
  • إطار TCF الخاص بـ IAB Europe — إطار الشفافية والموافقة لمعالجة البيانات
  • إرشادات EDPB بشأن تقليل البيانات — توجيهات أساسية للتعامل مع بيانات RDAP

موارد البنية التحتية للإنترنت​

توثيق السجلات​

  • توثيق Verisign RDAP — دليل التطبيق الرسمي لـ Verisign
  • توثيق ARIN RDAP — تفاصيل تطبيق RDAP الخاص بـ ARIN
  • توثيق RIPE NCC RDAP — توثيق خدمة RDAP الخاصة بـ RIPE
  • توثيق APNIC RDAP — تفاصيل تطبيق APNIC
  • توثيق LACNIC RDAP — موارد RDAP الخاصة بـ LACNIC

أدوات تحليل البروتوكول​

  • محلل RDAP الخاص بـ Wireshark — تحليل بروتوكول الشبكة لحركة مرور RDAP
  • DNSViz — أداة تصور وتحليل DNS لتشخيص الأخطاء
  • أداة MTR لتشخيص الشبكة — تحليل مسار الشبكة لمشاكل الاتصال

أدوات المطور والمكتبات​

مكتبات وعملاء RDAP​

  • rdap-rs (Rust) — عميل RDAP عالي الأداء بلغة Rust
  • python-rdap (Python) — مكتبة عميل RDAP بلغة Python
  • go-rdap (Go) — تطبيق Go مع دعم موسع للسجلات
  • rdap-cli (سطر الأوامر) — عميل RDAP لسطر الأوامر للكتابة البرمجية

أدوات اختبار الأمان​

  • ZAP (وكيل هجوم Zed) — اختبار أمان تطبيقات الويب مع مسح نقاط نهاية RDAP
  • Bandit (محلل أمان Python) — محلل أمان لتحليل كود Python
  • Semgrep — أداة تحليل ثابت مع قواعد مخصصة لأمان RDAP
  • Trivy — فحص الحاويات ونقاط ضعف التبعيات

موارد الامتثال والقانون​

الأطر التنظيمية​

  • سياسة بيانات التسجيل الخاصة بـ ICANN — السياسة الرسمية التي تحكم بيانات WHOIS/RDAP
  • موارد خصوصية لجنة التجارة الفيدرالية — توجيهات الخصوصية الصادرة عن لجنة التجارة الفيدرالية الأمريكية
  • مجلس حماية البيانات الأوروبي — التوجيه الرسمي لـ GDPR وقراراته في الاتحاد الأوروبي
  • قضايا تنفيذ CCPA — إجراءات تنفيذية لمدعي عام كاليفورنيا

المعايير الصناعية​

  • ISO/IEC 27001:2022 — معايير إدارة أمن المعلومات
  • إطار الأمان السيبراني لـ NIST — إطار شامل للأمان السيبراني
  • ضوابط CIS الإصدار 8 — ضوابط الأمان الحرجة للأنظمة المواجهة للإنترنت
  • معايير خدمة SOC 2 — متطلبات التحكم في منظمات الخدمات

الأداء والتشغيل​

أدوات اختبار الأداء​

  • autocannon — أداة قياس أداء HTTP لاختبار نقاط نهاية RDAP
  • wrk — أداة حديثة لقياس أداء HTTP مع كتابة Lua
  • k6 — أداة اختبار الحمل المتمحورة حول المطور مع تكامل السحابة
  • Prometheus + Grafana — مجموعة المراقبة وإمكانية المراقبة لنشر RDAP الإنتاجي

أدوات DNS والشبكة​

  • DNSPerf — إطار اختبار أداء DNS
  • dnspython — مجموعة أدوات DNS لـ Python مع أمثلة تكامل RDAP
  • MTR — أداة تشخيص شبكة تجمع بين ping وtraceroute
  • nmap — أداة اكتشاف الشبكة وتدقيق الأمان

المجتمع والتعلم​

منتديات النقاش والمجتمعات​

  • مجموعة عمل IETF REGEXT — مجموعة عمل IETF الرسمية لتطوير بروتوكول RDAP
  • المجتمع التقني لـ ICANN — نقاشات تقنية حول تطبيق RDAP وسياساته
  • Reddit r/netsec — مجتمع أمان الشبكات مع نقاشات RDAP
  • Matrix #rdapify:matrix.org — قناة الدردشة الجماعية لـ RDAPify

المؤتمرات والفعاليات​

  • اجتماعات IETF — اجتماعات IETF الرسمية مع جلسات مجموعة عمل RDAP
  • ورش عمل DNS-OARC — ورش عمل مركز عمليات DNS والتحليل والأبحاث
  • DEF CON — مؤتمر الأمان مع مسار أمان البنية التحتية
  • Black Hat — مؤتمر الأمان مع تركيز على أمان الشبكات والبروتوكولات

الأبحاث والأوراق الأكاديمية​

أبحاث RDAP​

  • "RDAP: بروتوكول WHOIS الجديد" (IEEE Communications Magazine) — تحليل أكاديمي لتصميم بروتوكول RDAP
  • "تطبيقات RDAP الحافظة للخصوصية" (ACM TOPS) — بحث حول تقنيات الخصوصية لعملاء RDAP
  • "DNSSEC وRDAP: تقنيتا أمان تكمّلان بعضهما" (RFC Editor) — تحليل تقني لتكامل DNSSEC وRDAP

أبحاث الأمان​

  • "هجمات SSRF في تطبيقات الويب الحديثة" (USENIX Security) — بحث هجمات SSRF ذو صلة بعملاء RDAP
  • "كشف البيانات الشخصية في البيانات المنظمة وغير المنظمة" (IEEE) — بحث أكاديمي حول تقنيات كشف البيانات الشخصية
  • "استراتيجيات التخزين المؤقت للأنظمة الموزعة جغرافياً" (ACM) — بحث حول التخزين المؤقت الموزع للخدمات العالمية

الشرح والتعلم​

أدلة عملية​

  • RDAP Protocol Deep Dive (Cloudflare Blog) — دليل تطبيق عملي مع أمثلة
  • "بناء عملاء RDAP آمنين" (OWASP Foundation) — دليل تطوير عميل RDAP المتمحور حول الأمان
  • "معالجة البيانات المتوافقة مع GDPR" (DPO Handbook) — توجيه شامل لمسؤولي حماية البيانات
  • ورشة عمل تطوير عميل RDAP — مستودع GitHub مع مواد ورش العمل والتمارين
