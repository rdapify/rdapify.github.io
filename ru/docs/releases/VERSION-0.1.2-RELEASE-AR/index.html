ملاحظات إصدار RDAPify v0.1.2

تاريخ الإصدار: 27 يناير 2026
الإصدار: 0.1.2
الحالة: ✅ جاهز للإنتاج

🎉 ما الجديد​

ساحة التجربة التفاعلية​

يتضمن RDAPify الآن تجربة "جرّب قبل التثبيت" جاهزة للإنتاج ويمكن الوصول إليها على rdapify.com/playground.

الميزات الرئيسية​

  1. إدارة معرّف العميل

    • تحديد مستقر للمتصفح باستخدام crypto.randomUUID()
    • تخزين دائم في localStorage (rdapify_client_id)
    • بديل للمتصفحات القديمة: Date.now() + Math.random()

  2. تتبع الحصة في الوقت الفعلي

    • عرض الاستعلامات اليومية المتبقية (remainingToday)
    • إظهار وقت إعادة تعيين الحصة (resetAt)
    • تعطيل الزر تلقائيًا عند نفاد الحصة

  3. معالجة حد المعدل

    • استجابات خطأ 429 بشكل سلس
    • رسالة "تم الوصول للحد اليومي" سهلة الاستخدام
    • تلميح Retry-After: "حاول مرة أخرى خلال X دقائق"

  4. دعم مديري الحزم المتعددة

    • أمر npm install مع زر النسخ
    • أمر yarn add مع زر النسخ
    • أمر pnpm add مع زر النسخ

  5. التكامل مع الموقع

    • الوصول إلى الساحة من القائمة الرئيسية
    • تكامل سلس مع Docusaurus عبر iframe
    • تصميم متجاوب لجميع أحجام الشاشات

تحسينات جودة الكود​

  • إصلاحات ESLint: تم حل 6 مشاكل (4 أخطاء + تحذيرين)
    • إصلاح المتغيرات غير المستخدمة في Logger.ts
    • إصلاح تأكيدات النوع في enhanced-validators.ts

📦 التثبيت​

# باستخدام npm
npm install rdapify

# باستخدام yarn
yarn add rdapify

# باستخدام pnpm
pnpm add rdapify

🚀 جرّبه الآن​

قم بزيارة rdapify.com/playground لتجربة RDAPify بدون تثبيت أي شيء!

الميزات:

  • استعلام عن النطاقات وعناوين IP وأرقام ASN
  • رؤية استجابات RDAP الحقيقية
  • تتبع حصتك اليومية
  • نسخ أوامر التثبيت بنقرة واحدة

📝 التفاصيل التقنية​

بنية الساحة​

الواجهة الأمامية (ثابتة):

  • playground/public/index.html - الواجهة الرئيسية
  • playground/public/app.js - منطق العميل مع تتبع الحصة
  • playground/public/style.css - التنسيق المتجاوب

الخلفية (الإنتاج):

  • GitHub Pages: استضافة الملفات الثابتة
  • Cloudflare Worker: نقاط نهاية API (/api/*)
    • GET /api/health - فحص الصحة
    • POST /api/query - استعلامات RDAP مع تحديد المعدل

التطوير المحلي:

  • playground/api/proxy.js - وكيل Express (للتطوير فقط)
  • يحل مشاكل CORS أثناء الاختبار المحلي

تنسيق استجابة API​

{
  "success": true,
  "data": { /* استجابة RDAP */ },
  "remainingToday": 3,
  "resetAt": "2026-01-28T00:00:00.000Z"
}

تحديد المعدل​

  • الحصة اليومية: 5 استعلامات لكل معرّف عميل
  • حد المعدل: استعلامان في الدقيقة
  • استجابة 429: تتضمن رأس Retry-After
